# 小红书内容周批量确认工具 · 功能说明

## 概述

面向小红书账号的内容批量生产与自动发布工具。运营人员每周只需完成一次6步流程，即可批量完成选题、转写、底图、视频、文案的全链路生产，并按计划自动发布。全流程各环节提示词均可灵活调整，支持逐条重新生成，兼顾效率与可控性。

---

## 首页（运营中心）

系统入口，提供全局视角。

**账号切换**
顶栏支持在小红书、抖音、视频号三个账号间切换，当前账号以绿点标识在线状态。

**草稿恢复**
若有未完成的批量确认流程，页面顶部自动出现黄色提示条，显示中断时所在步骤、已选篇数、上次保存时间，可直接「继续编辑」或「丢弃草稿」。

**本周入口**
- 已创建：展示本周7篇内容列表（封面色块、标题、发布时间、点赞/评论数、状态），可直接进入编辑；已发布条目显示「查看数据」，定时排队条目显示「修改时间」，发布失败条目显示「重新发布」。
- 未创建：显示可用对标内容数量、已生成底图数，点击进入批量确认流程。

**下周入口**
常驻展示下周日期范围及当前可用对标内容数量，可随时提前启动下周内容规划。

**数据概览**
四格统计卡显示：累计发布篇数、本周待发布数、定时排队数、监控账号数。

**历史内容**
按周折叠展示所有历史内容，每周标题行汇总已发布/定时中/失败数量。支持按状态筛选（全部 / 已发布 / 定时中 / 发布失败），展开后逐条显示互动数据和操作按钮。

## 批量确认流程

### 概述

面向「心理学高知女性博主」账号的内容生产辅助工具。运营人员每周通过6个步骤，完成从选题到发布计划的全流程确认，最终导出调度器所需的 JSON 文件自动发布。

**流程总览：选对标内容 → 转写检查 → 选底图 → 检查视频 → 确认文案 → 完成**

每步完成后方可进入下一步，已完成的步骤可随时跳回修改。进度自动保存至浏览器本地，刷新不丢失。

---

### 步骤一 · 选对标内容

- 展示上周从10个对标账号抓取的内容库（共约18条）
- 每条内容显示：账号、标题、点赞/评论/收藏数、时长、热度分、人设契合度分
- 支持关键词搜索、按账号筛选、多维度排序（推荐值/最新/时长/互动数）
- 可展开查看原始文字稿
- 从内容池选7条分配至本周7天，支持调整天序、移除重选
- 选满7条自动解锁下一步

### 步骤二 · 转写内容检查

- 顶部展示转写提示词，支持展开编辑、保存
- 每条内容左右对比显示：对标原文 ｜ AI 转写文案
- 转写文案可直接编辑，实时字数统计
- 每条可单独触发重新生成；顶部按钮支持全部重新生成

### 步骤三 · 选底图

- 顶部展示底图生成提示词，支持编辑保存
- 每次调用接口生成4张底图，可累计多批后再挑选
- 从图库中选7张（点击选中，再点取消），支持调整天序
- 底图可 hover 后点放大镜图标查看大图（不影响选图操作）

### 步骤四 · 检查视频

- 分别配置 5B 模型和 14B 模型的视频生成提示词
- 每条视频可单独指定使用 5B 或 14B 模型
- 进入步骤时自动为每条内容预分配唯一视频 ID（格式：`VID-YYYYMMDD-NNN`），可点击复制
- 支持一键生成全部视频，也可逐条单独生成/重新生成
- 视频生成后在卡片内展示播放器，逐条确认后勾选
- 显示整体进度（已确认 N/7）

### 步骤五 · 确认文案

- 顶部并排展示三个提示词面板：标题生成、封面生成、文案&SEO，均可独立编辑
- 每条内容显示：封面预览、标题（20字限制）、详情文案、SEO标签、发布时间
- 封面点击可放大查看
- 标题、详情文案、SEO标签各有内嵌刷新图标，逐条重新生成，互不影响
- 封面支持逐条重新生成
- 视频 ID 在每张卡片顶部展示，可点击复制
- 所有字段均可手动编辑

### 步骤六 · 完成

- 汇总展示本周7天的发布时间计划
- 一键导出 JSON 文件，包含每天的视频 ID、标题、封面、详情文案、SEO 标签、发布时间，供调度器直接使用

---

## 通用交互

- **提示词管理**：每类提示词（转写/底图/视频5B/视频14B/标题/封面/文案）独立存储，修改后保存生效，下次打开保留
- **草稿自动保存**：所有操作结果实时存入 localStorage，600ms 防抖
- **大图预览**：步骤三底图、步骤五封面均支持点击放大，Esc 或点击背景关闭
